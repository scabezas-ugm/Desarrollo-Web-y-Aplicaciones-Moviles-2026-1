# Programa del Curso

## Descripción
La asignatura Desarrollo Web y Aplicaciones Móviles tiene como objetivo dotar a los estudiantes de conocimientos teóricos y prácticos para diseñar, desarrollar e implementar aplicaciones web y móviles utilizando tecnologías modernas. A lo largo del curso, se abordan temas como los fundamentos del desarrollo web y móvil, el uso avanzado de HTML, CSS y JavaScript, y la introducción a frameworks. 

Además, se abordan principios de diseño de interfaces y experiencia de usuario (UI/UX). 

El curso culmina con un proyecto integrador donde los estudiantes aplican lo aprendido para crear y desplegar aplicaciones web y móviles completas, enfocándose en la accesibilidad, el rendimiento y la seguridad.

## Resultados del perfil de egreso
I.a. Evalua servicios y sistemas que satisfagan todos los requisitos del usuario y se comporten de forma fiable, eficiente y segura, sean asequibles de desarrollar y mantener, cumpliendo las normas de calidad, aplicando las teorías, principios, métodos y prácticas de la Ingeniería del Software.

I.b. Diseña modelos de seguridad de los sistemas de las organizaciones que permitan el desarrollo de la automatización, a partir de la revisión de informes informáticos, utilizando soluciones de software y hardware, e integrando sistemas inteligentes a la estructura de las organización y estrategias operativas, junto con la efectividad del manejo y disposición de la información.

II.b. Evalúa las topologías operativas, los recursos tecnológicos, junto con la efectividad del manejo y disposición de la información, para establecer los protocolos de seguridad de los sistemas las redes organizacionales, que le permitan la implementación de proyectos informáticos que reduzcan los riesgos en estas áreas.

V.a. Reconocer las dimensiones culturales, identificando su relevancia para la comprensión de la cultura y del ser humano.

## Unidades de Aprendizaje

* Unidad 1 (25%): Diseño de Wireframes / Mockups
* Unidad 2 (25%): Implementación Frontend
* Unidad 3 (25%): Desarrollo Backend y Consumo de Servicios
* Tareas, Controles y/o Actividades (25%)
